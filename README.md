## О проекте

`nameless` - неофициальный iOS-клиент на базе upstream-кода Telegram с собственным брендингом, тёмным визуальным стилем, округлыми элементами, glass-поверхностями и отдельным центром настроек nameless.

Название `nameless`, логотип, бейдж, оформление, скриншоты и все проектные доработки являются защищёнными материалами проекта. Upstream-код и сторонние зависимости сохраняют свои оригинальные лицензии.

## Что уже работает

- Вкладка `nameless` в настройках с кастомной круглой иконкой.
- Центр настроек nameless с разделами `Внешний вид`, `Режим призрака`, `О nameless`, `Функции nameless`, `Жидкое стекло`.
- Поиск по настройкам nameless.
- Каталог `Поиск настроек`, который открывает функции по нажатию.
- Скрытые legacy-вкладки открываются долгим нажатием на `Возможности Telegram`.
- Ссылки `Канал nameless`, `Разработчик glswee`, `Stiven VPN` открываются внутри клиента.
- Liquid Glass-переключатели для сообщений, настроек, профиля, подарков и inline-кнопок.
- Видео на фоне чата: выбор, замена, удаление, включение и выключение зацикленного видео без звука.
- Круглые icon-only кнопки профиля без текстовых подписей.
- Официальный блок разработчика nameless: статус, карточка и verification footer.
- В видимых селекторах и бейджах используются только nameless-ассеты.
- Подключены модули: режим призрака, локальный premium, экспорт чата, фейковая геолокация, пароль чата, double bottom, голосовые инструменты и supporters.

## Функции nameless

Ниже перечислены реальные функции, которые уже подключены в клиенте. Короткое описание рядом с каждой функцией показывает, что она делает.

### Центр nameless

- `Поиск настроек` - ищет все функции nameless и открывает нужный экран.
- `Внешний вид` - раздел для редизайна интерфейса, карточек, фона, профиля и музыки.
- `Режим призрака` - раздел приватных переключателей для скрытия статусов и действий.
- `О nameless` - карточка проекта, ссылки на канал, разработчика и VPN.
- `Жидкое стекло` - управление glass-оформлением для основных экранов и элементов.

### Сообщения

- `Сохранять удалённые сообщения` - сохраняет удалённые сообщения локально.
- `Сохранять медиа удалённых сообщений` - оставляет фото, видео и файлы из удалённых сообщений.
- `Сохранять реакции удалённых сообщений` - хранит реакции в локальной базе.
- `Сохранять удалённые сообщения для ботов` - отдельно сохраняет удалённое из диалогов с ботами.
- `Сохранять историю редактирования` - хранит предыдущие версии сообщений.
- `Редактировать сообщения собеседника` - позволяет править входящие сообщения локально на устройстве.
- `Сохранять самоуничтожающиеся сообщения` - сохраняет self-destruct media, если это доступно на клиентском уровне.
- `Сохранять самоуничтожающиеся медиа` - сохраняет исчезающие вложения локально.
- `Скрыть детекцию скриншотов` - убирает реакцию клиента на факт скриншота там, где это возможно.
- `Убрать blur секретных чатов` - отключает размывание в secret chat при просмотре.
- `Сохранять историю чатов` - хранит локальную историю диалогов.
- `Восстановить сохранённые чаты` - возвращает ранее сохранённые чаты.
- `Показывать удалённые сообщения` - отображает удалённые сообщения, если они уже были сохранены.
- `Оригинал отредактированных` - показывает исходный вариант сообщения до редактирования.
- `Сокращать сообщения` - включает упрощённый вид длинных сообщений.
- `Счётчик символов при вводе` - показывает длину текста во время набора.
- `Счётчик символов в сообщениях` - показывает счётчик длины уже отправленного сообщения.

### Режим призрака

- `Скрыть онлайн-статус` - не показывает статус присутствия.
- `Скрыть статус набора` - скрывает typing indicator.
- `Скрыть статус записи голосового` - маскирует запись голосового сообщения.
- `Скрыть статус загрузки файлов` - маскирует отправку файлов.
- `Скрыть статус чтения сообщений` - отключает read receipts.
- `Скрыть просмотр историй` - отключает отметку просмотра stories.
- `Читать при действиях` - уменьшает видимость действий в чате.
- `Отложенная отправка` - задерживает отправку сообщений на выбранное время.
- `Устройство` - включает поведение, связанное с устройством в приватных сценариях.
- `Подмена геолокации` - подставляет выбранные координаты.
- `История онлайна` - показывает историю присутствия, если доступно.
- `Очистить историю онлайна` - удаляет локальную историю присутствия.
- `Всегда онлайн` - оставляет аккаунт доступным как онлайн в приватной логике клиента.
- `Скрыть статус набора текста` - дополнительный приватный фильтр для typing.
- `Скрыть просмотр истории онлайна` - скрывает действия, связанные с историей просмотра.

### Конфиденциальность и безопасность

- `Сохранять защищённый контент` - даёт доступ к защищённому контенту там, где это поддержано клиентской частью.
- `Отключить рекламу` - убирает рекламу в поддерживаемых местах.
- `Скрыть спонсора прокси` - убирает подпись sponsor у прокси.
- `Защита от мошенников` - добавляет дополнительные предупреждения и проверки.
- `Предупреждать перед звонком` - показывает предупреждение до звонка.
- `Скрыть номер телефона` - убирает номер из профиля там, где это поддерживается.
- `Показывать ID и DC в профиле` - добавляет служебные данные в профиль.
- `Показывать приблизительную дату` - включает упрощённое отображение дат.
- `Обход защищённого контента` - открывает защищённые элементы, если клиентская логика это позволяет.

### Внешний вид

- `Полные числа вместо сокращения` - показывает `1400` вместо `1.4K`.
- `Убирать Zalgo` - очищает искажённый текст.
- `Локальный баланс звёзд` - показывает локально заданный баланс Stars.
- `Панель форматирования над клавиатурой` - держит панель форматирования видимой.
- `Кнопка "Перевести" всегда видима` - не прячет кнопку перевода.
- `Круглые кнопки в профиле` - делает профильные кнопки icon-only без текста.
- `Стиль карточки музыки` - включает альтернативный вид музыкальной карточки.
- `Видео на фоне чата` - ставит зацикленный беззвучный фон-видео для экранов.
- `Обои профиля` - отдельный фон или обложка для профиля.
- `Фейковая геолокация` - визуальная подмена координат в интерфейсе.
- `Замена шрифта` - использует выбранный пользовательский шрифт.
- `Плавающие glass-элементы` - подключают Liquid Glass-оформление к карточкам, кнопкам и панелям.

### Дополнительно

- `Экспорт чата` - экспортирует историю диалога.
- `Неограниченные избранные стикеры` - снимает лимит на favourites.
- `Неограниченное закрепление чатов` - расширяет лимит pinned chats.
- `Свайп-опции чатов` - меняет действия на свайпе.
- `Папки снизу` - переносит папки в нижнюю часть интерфейса.
- `Скрыть кнопку поиска` - убирает кнопку поиска из верхней панели.
- `Скрыть кнопку записи голоса` - убирает кнопку записи голоса.
- `Скрыть нижний таббар` - скрывает нижнюю навигацию, если это нужно.
- `Включить пользовательский шрифт` - использует выбранный шрифт проекта.
- `Виджеты и плагины` - подключает дополнительные модули и расширения.
- `Управление плагинами` - экран для работы с плагинами.
- `Скопировать диагностику` - копирует техническую диагностику клиента.
- `Перевод сообщений через Google` - переводит сообщения через внешний сервис.
- `Переводить текст из меню отправки` - переводит текст прямо из меню.
- `Транскрипция голосовых через AI` - распознаёт голосовые сообщения.
- `API-ключи сервисов` - хранит ключи для подключаемых сервисов.
- `Whitegram AI` - подключаемый AI-блок интерфейса.
- `Кроссфейд` - плавное смешивание музыки в плеере.
- `Эквалайзер` - настройка звука в плеере.

### Функции со звёздочкой

Звёздочка `*` в интерфейсе и README означает, что функция частично реализована, находится в разработке или подключена как заглушка.

- `Смена голоса *` - экран есть, но мост к полному обработчику голоса ещё не подключён полностью.

## Статус реализации

- Брендинг и видимые строки переведены на `nameless`.
- Логотип с чёрным персонажем подключён как app/settings icon.
- Горизонтальный `namelessBadge` подключён для README и badge UI.
- GitHub Actions собирает unsigned iOS artifact через Bazel на macOS runner.
- Liquid Glass покрытие расширяется постепенно: ключи и настройки уже подключены, часть поверхностей использует существующие glass/fallback-компоненты.
